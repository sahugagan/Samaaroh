Cinematic media files required:

1) hero-3d.mp4
   - Used as the hero cinematic background video
   - Path: /media/hero-3d.mp4

2) hero-poster.jpg
   - Fallback poster image while video is loading
   - Path: /media/hero-poster.jpg

3) ambient.mp3
   - Ambient audio file for Sound ON toggle
   - Path: /media/ambient.mp3

If these files are missing, cinematic visuals/audio will not appear.
Place files exactly in: client/public/media/
